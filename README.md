# PixelWeb.ge РђЊ рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃАрЃћрЃарЃЋрЃўрЃА-рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃАрЃЌрЃЋрЃўрЃА. рЃАрЃгрЃарЃљрЃцрЃў, рЃАрЃБрЃцрЃЌрЃљ рЃЊрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃўрЃџрЃў рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљ Next.js 14-рЃўрЃАрЃљ рЃЊрЃљ Tailwind CSS-рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ.

## ­Ъџђ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў

- рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЊрЃўрЃќрЃљрЃўрЃюрЃў Tailwind CSS-рЃўрЃЌ рЃЊрЃљ рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃўрЃџрЃў UI рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃЌ
- рЃАрЃарЃБрЃџрЃљрЃЊ рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў РђЊ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў, рЃбрЃљрЃЉрЃџрЃћрЃбрЃў, рЃЊрЃћрЃАрЃЎрЃбрЃЮрЃърЃў
- рЃњрЃџрЃБрЃЋрЃў рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў Framer Motion-рЃўрЃЌ (рЃњрЃљрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ prefers-reduced-motion)
- Dark/Light рЃЌрЃћрЃЏрЃљ РђЊ ThemeProvider рЃЊрЃљ рЃЦрЃБрЃџ UI ThemeToggle
- рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃерЃарЃўрЃцрЃбрЃў РђЊ BPG ExtraSquare Mtavruli (app/globals.css)
- SEO рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ РђЊ Next.js Metadata API
- рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃљ РђЊ @vercel/analytics
- рЃърЃћрЃарЃцрЃЮрЃарЃЏрЃљрЃюрЃАрЃў РђЊ Next.js App Router

## ­ЪєЋ рЃЉрЃЮрЃџрЃЮ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃћрЃЉрЃў

- Image Slider: рЃЦрЃЋрЃћрЃЊрЃљ gradient overlay рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃЎрЃЮрЃюрЃбрЃљрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА
- Scroll-to-top рЃЏрЃЮрЃЉрЃФрЃюрЃћ рЃдрЃўрЃџрЃљрЃЎрЃў
- PageTransition: reduced motion рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ
- Footer: рЃќрЃћрЃЊрЃљ рЃЏрЃ«рЃљрЃарЃћрЃА рЃЏрЃЎрЃарЃЌрЃљрЃџрЃў gradient divider
- Home Hero: рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃўрЃБрЃџрЃў gradient orbs рЃцрЃЮрЃюрЃўрЃА рЃАрЃўрЃдрЃарЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА

## ­Ъїљ API рЃарЃЮрЃБрЃбрЃћрЃЉрЃў

  - `RESEND_API_KEY` (рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ Dev рЃарЃћрЃЪрЃўрЃЏрЃерЃў)
